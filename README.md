# Organizador académico y personal

Aplicación en Streamlit para organizar asignaturas, evaluaciones, horarios, sesiones de estudio, metas personales y revisiones semanales.

## Ejecutar localmente

```bash
pip install -r requirements.txt
streamlit run app.py
```

La base de datos SQLite se crea automáticamente como `organizador.db` al iniciar la app.
